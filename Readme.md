# AutoForge Supply Chain Analytics

**Automotive Supply Chain Analysis Project**  
Real-world messy ERP data → Cleaning → Analysis → Business Insights

## Project Overview
You are the new Supply Chain Analyst at **AutoForge Motors** (SUVs & Sedans manufacturer).  
This repo shows how to handle messy data from ERP, MES, and TMS systems and turn it into actionable insights to prevent production line stoppages.

## Day 1: Google Sheets - Taming Messy Master Data
- Generated realistic messy automotive data (suppliers, BOM, inventory, logistics, demand)
- Cleaned data using Trim whitespace, Remove duplicates, and proper formatting
- Linked tables using **XLOOKUP**
- Created **Master Combined View** with first KPIs:
  - Lead time risk
  - Critical components
  - Defect impact

## Folder Structure
- `data/raw/` → Original messy ERP exports
- `data/cleaned/` → Cleaned files + master_combined.csv
- `notebooks/` → Python scripts (coming soon)
